## 1. 游戏定位

### 一句话定位
> 一款治愈系连连看手游——山田律整理岛上散落的记忆碎片，把一座沉睡的日式小岛，一点一点唤醒。

### 三句话展开
Glow Island 是以步数制连连看为核心机制的休闲手游，玩家扮演离开东京广告公司、逃到輝島（Kagayaki-jima）的创意总监律，通过连接同色图块积累光能，整理散落在岛屿各处的记忆碎片，驱动小岛六片区域从荒芜逐步恢复为生机盎然的日式岛屿风貌。游戏无体力系统、无倒计时压力、无强制社交义务，以真正的自由游玩节奏兑现治愈承诺，并通过精心打磨的渐进式视觉修复动画和温情叙事提供竞品无法复制的情感成就感。动物之森式的柔和美术风格和"整理记忆·修复连接"的核心叙事，让每一次消除都成为律和这座岛相互疗愈的小小瞬间——而他与丸山ひなた之间那条若有若无的情感线，让温柔多了一层余温。

### 品类定位
- **品类**：治愈系休闲连连看（Healing Casual Tile-Connect）
- **平台**：iOS / WebGL
- **目标用户**：25-40岁，偏好低压力放松体验的都市休闲玩家；曾体验过工作倦怠、渴望「慢下来」的情感共鸣群体；男女玩家均为核心受众——律的职场倦怠叙事提供普遍共鸣，ひなた的慢热情感线拓宽了女性玩家的情感投入维度
- **情感关键词**：温柔、治愈、成就感、慢下来、自由
- **参考氛围**：どうぶつの森（动物之森）— 柔和粉彩色调、圆润友好造型、岛屿生活的细腻温情

## 2. 核心体验目标

基于自我决定理论（Deci & Ryan），Glow Island 的三重内在动机设计：

| 动机维度 | 设计响应 |
|----------|----------|
| **自主（Autonomy）** | 无体力门控、随时进出、禅境模式可自由选择、季节内容免费开放 |
| **胜任（Competence）** | 平滑难度曲线、连击反馈系统、4种特殊块赋予策略感、失败不惩罚资源 |
| **关联（Relatedness）** | 律与岛民（健三、梅子、夏帆、冬子、ひなた）的温情叙事、灯塔守渉的记忆日志情感积累、律与ひなた之间那条若有若无却真实存在的情感线、修复动画带来的"我让它改变了"的归属感 |

## 3. 核心循环（Core Loop）

> 局内时长：30秒 - 3分钟/关
>
> 叙事框架：每一局关卡，都是律在岛上整理散落记忆的一个小片段——翻开一只旧箱子、理清一捆渔网、拨开花田里的杂草。棋盘上的彩色图块是记忆的碎片，连接同色、让它们归位，就是在帮这座岛找回遗失的故事。

```
进入关卡（律翻出一样旧物 / 走进一处荒芜的角落）
    ↓
观察棋盘布局（图块颜色/位置/特殊块分布）
    ↓
规划连接路径（同色记忆碎片连线，路径可弯折，最短2个）
    ↓
触发消除 → 即时反馈（音效 + 粒子特效 + 连击计数）
    ↓
连击积累 → 触发特殊块生成/光能加成
    ↓
消除目标图块 / 达成关卡目标
    ↓
关卡结算（星级评定 / 材料掉落 / 光能积累）
    ↓
返回輝島世界地图
```

### 核心循环设计原则
- **步数制，非时间制**：每关有步数上限，压力来自策略而非焦虑——就像律在岛上慢慢生活，没有截止日期
- **连线自由度**：连线路径可弯折绕行（非直线限制），增加策略空间
- **即时反馈**：每次连接触发专属音效和粒子效果，形成上瘾性微奖励循环
- **失败宽容**：步数耗尽仅回到关卡起点，不扣材料、不消耗体力——岛不会因为你慢一点就放弃你
- **连击奖励**：连续消除同色图块触发连击乘数，3连击以上生成随机特殊块

## 4. 元循环（Meta Loop）

> 局外周期：单次修复动画30秒 - 2分钟，区域完整修复跨越数天游玩

```
通关关卡
    ↓
获得材料掉落（浮木/绳索/花种/陶片/温泉石等，对应章节主题）+ 光能积累
    ↓
材料自动轻合成（无复杂配方，系统自动聚合同类材料）
    ↓
区域修复进度条推进
    ↓
达到阶段阈值 → 触发渐进式修复动画（4阶段之一）
    ↓
Glow图块光能积累满阈值 → 触发区域"点亮"视觉特效
    ↓
章节完整修复 → 解锁 NPC（健三/梅子/夏帆/冬子）标志性时刻动画 + 丸山渉记忆日志碎片 + 下一章节入口
    ↓
解锁新关卡包 → 新图块种类 / 新棋盘主题 / 新障碍类型
```

### 元循环设计原则
- **进度可感知**：每次关卡结算后立即可见輝島的变化，无需等待
- **轻量无负担**：材料系统无库存管理压力，满包自动合成，不造成选择疲劳
- **情感奖励密度**：每4-8个关卡触发一次可见的视觉变化，确保玩家长期有期待感
- **叙事驱动进度**：每个修复阶段与岛民行为/环境细节深度绑定，同步解锁渉的记忆日志；进度不只是数字，是律与这座岛一同愈合的轨迹——也是他与ひなた之间那条细线，一点一点变得清晰的过程

### 章节情绪递进逻辑
```
到来·悠闲（抵达/第一次深呼吸/律与ひなた码头错身）
    → 扎根·温暖（安定/「这里是我待的地方」/ひなた偶尔出现帮一把）
    → 感受·治愈（共鸣/眼泪来得莫名其妙/ひなた带路，讲祖父的树）
        → 栖息·安静（内省/律独处/他想起ひなた说的那句话）
            → 接受·放松（接纳过去/不再愤怒/律与ひなた的钓鱼时光）
                → 点亮·完成（律与岛，互相唤醒/ひなた站在光里，松开了那枚戒指）
```

### 各章节视觉色调（动物之森式柔和粉彩）
- **第1章**：淡蓝+米白，阳光斜射，明亮但饱和度低；海浪拍打木栈道的泡沫是浅白色
- **第2章**：暖黄+橙褐，午后阳光打在石板路上；梅子的陶窑是朱红色的砖
- **第3章**：粉黄+草绿，漫山野菊与油菜花，薄雾清晨感；夏帆的草帽是麦秸黄
- **第4章**：深绿+棕褐，树荫斑驳，晚风带凉意；篝火的橙色是整章唯一的暖点；律的日记本封面是旧牛皮色，在火光里翻动
- **第5章**：珍珠白+雾蓝，水雾弥漫，矿物质气息；温泉水面是会动的浅青色
- **第6章**：深蓝夜色+暖黄灯光，一束光切开黑暗；灯塔光柱是整个游戏最亮的颜色

## 6. 特殊系统规格

### 6.1 四种特殊图块

| 图块名称 | 触发方式 | 效果 | 视觉表现 |
|----------|----------|------|----------|
| **炸弹块（Bomb Tile）** | 连接5个同色图块 | 消除以其为中心的3×3范围内所有图块 | 橙色闪光球体，激活时爆炸粒子效果 |
| **闪电块（Lightning Tile）** | 连接6个同色图块 | 消除同行或同列所有图块（激活时玩家选择方向） | 电弧纹理，激活时白色闪光条横扫 |
| **彩虹块（Rainbow Tile）** | 连接7个同色图块 或 特定关卡奖励 | 作为通配符与任意颜色图块连接匹配 | 七彩流光，消除时彩虹粒子爆发 |
| **Glow发光块（Glow Tile）** | 随机散布于关卡棋盘 | 连接时产生"光链"效果，光能奖励×3，触发相邻图块短暂发光 | 金色/彩色光晕脉冲，连接时光链动画流向岛屿 |

### 6.2 动态棋盘事件

特定关卡（每章约1/5的关卡触发）会随机发生棋盘动态事件，强化章节主题沉浸感：

| 章节 | 棋盘事件 | 效果描述 |
|------|----------|----------|
| 第1章 | 海浪涌上栈道 | 每3步，棋盘最底行随机"被海浪冲走"1个图块，需重新规划路径；健三在画面角落抬头看了一眼，继续钓鱼 |
| 第2章 | 梅子开窑 | 棋盘中随机1-2格亮起陶窑的橙色光晕，该格图块与任意相邻色连接时奖励步数+1 |
| 第3章 | 野菊飘落 | 每5步，随机1个位置飘落野菊图块，连接时额外掉落花种材料；夏帆在远处哼了一句曲子 |
| 第4章 | 萤火虫飞过 | 萤火虫短暂照亮被遮挡/障碍覆盖的图块颜色，持续2步；营地夜晚专属视觉 |
| 第5章 | 温泉水雾弥漫 | 棋盘中央3×3区域图块颜色被雾气遮住，需从周边区域推断颜色 |
| 第6章 | 灯塔光扫过 | 每8步，光柱扫过棋盘，照亮一整行图块并触发该行所有Glow块的光能奖励；冬子的剪影在灯塔台阶上一动不动 |

### 6.3 障碍系统

| 障碍类型 | 出现章节 | 清除方式 | 设计意图 |
|----------|----------|----------|----------|
| **锁链图块** | 第1章引入 | 连接该图块所在的消除路径2次 | 基础策略阻碍，引导玩家规划路径优先级 |
| **泥沼覆盖** | 第2章引入 | 连接覆盖格的图块消除 | 限制连线可用格数，逼迫玩家绕路规划 |
| **冰封图块** | 第3章引入 | 消除相邻图块3次解冻 | 引入"边际消除"策略，增加关卡维度 |
| **双色迷雾** | 第5章引入 | 使用彩虹块或同时连接两种颜色 | 高难度策略元素，仅出现于后期关卡 |

## 7. 关卡设计原则

### 7.1 难度曲线设计
- **前15关（教学阶段）**：每关引入1个新概念，不组合，给予充足步数
- **16-45关（成长阶段）**：组合机制，适度减少步数裕量，连击奖励设计明确
- **46-80关（挑战阶段）**：障碍组合，动态事件引入，部分关卡设计"完美路径"
- **80关后（内容阶段）**：复杂棋盘布局，多目标同时达成，禅境模式持续提供放松出口

### 7.2 关卡类型分布（每章15-20关）
- **故事关（25%）**：推进NPC叙事，步数宽松，以视觉奖励为主
- **普通关（55%）**：标准连线消除，阶梯难度，步数适中
- **挑战关（15%）**：限制步数+多障碍，通关奖励额外材料
- **禅境关（5%）**：每章2-3关，每日随机开放，无步数无失败

### 7.3 失败设计原则
- 失败后不弹出购买提示（连续失败2次内）
- 第3次失败后，以非侵入方式提示"是否使用助攻器"（场景化提示，非弹窗打断）
- 提供"跳过关卡"选项（消耗丹青石），但数量显示透明，杜绝隐藏定价

## 8. 音频与视觉方向

### 8.1 视觉设计方向
- **美术风格**：动物之森式柔和粉彩风——圆润的形体、低饱和的暖色系、让人看了就想揉一下的质感。恢复前灰调低饱和，恢复后逐步过渡到明亮的岛屿配色
- **角色造型**：律和岛民均采用圆头圆脑的Q版比例，表情简洁但情绪丰富；NPC各自有鲜明的标志性物件（健三的渔夫帽、梅子的靛蓝围裙、夏帆的草帽+野花、冬子的旧钥匙串、ひなた的木制发簪+胸前挂着的渔夫戒）
- **光影效果**：阳光总是"下午三点的温柔斜射"，夜晚以暖黄灯光为主；不追求写实，只追求温暖
- **修复动画**：每阶段修复动画时长8-15秒，采用Spine动画驱动，植物生长/水流/光效扩散；动画结束时总有一个小细节——一只猫出现了、晾衣绳上多了一件衣服
- **UI设计**：简洁克制，棋盘区域占屏幕70%以上，UI元素退至边缘，不遮挡沉浸感；UI配色跟随当前章节色调
- **禅境模式视觉**：饱和度进一步提升，边框用光晕替代线条，整体柔化处理；像岛上最好的午后

### 8.2 音频设计方向
- **环境音分层**：海浪声（底层持续）+ 章节特色音（鸟鸣/风铃/篝火/水声/三味线远音）+ 偶发音效（NPC日式喃喃自语音效，无完整语音）
- **消除音效**：连击递进音调，1连击清脆单音→5连击和弦→10+连击加入弦乐收尾；整体音调偏木质/陶瓷质感，不用电子声
- **Glow图块专属音效**：连接Glow图块时加入高频晶体共鸣音，光链动画配合音效传导感，像灯塔的光脉冲
- **修复动画音乐**：每次区域修复触发专属30秒情感配乐，基调随章节情绪变化（悠闲→温暖→治愈→安静→放松→点亮）；第6章灯塔点亮时配乐加入风铃+低鸣弦乐
- **禅境模式音乐**：环境音+慢板钢琴，BPM约60，无强拍，营造漂浮感；像在輝島的清晨独自散步

## 9. 商业化策略概述

> 核心原则：**不P2W，只买便利和外观；体验优先，付费锦上添花**

### 9.1 双货币体系
- **沙滩币（Beach Coins）**：软货币，关卡通关/每日任务/活动掉落，用于普通助攻器和部分装饰
- **丹青石（Glowstone）**：硬货币，付费购买或少量活动奖励，用于续命/跳关/高级道具/限定装饰

### 9.2 IAP产品目录
| 产品名称 | 内容 | 定位 |
|----------|------|------|
| 小光晶包 | 6丹青石 | 低门槛入门，试水付费 |
| 中光晶包 | 30丹青石 | 主力单次购买包 |
| 大光晶包 | 88丹青石 | 高价值玩家单次大额 |
