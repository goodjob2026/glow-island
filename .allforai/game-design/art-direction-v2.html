1. 风格哲学 Style Philosophy

动物森友会式的温柔日式离岛：圆润、手绘、柔暖，随修复进度从褪色枯岛逐步"亮起"的治愈世界。

为什么选择 Animal Crossing 风格

Animal Crossing 的视觉语言天然降低认知负担：每个物件轮廓简洁、颜色区分鲜明、信息密度低而情感密度高。玩家在直觉层面就能识别图块类型，不需要"读"图块——只需"感受"它。这与 Glow Island 的消除玩法高度契合：快速连接靠的是图形识别，而非文字逻辑。

三项核心原则

  • 圆润优先：所有形状使用大圆角（≥16px），避免尖角。角色头身比接近 1:1.5 的 Q 版比例，四肢短而圆。
  • 手绘质感：轮廓线保留轻微不规则感（模拟铅笔起稿），主色块用单色平铺＋顶部极淡高光，整体如水彩图鉴插画风格。
  • 暖色主导：整体色温偏暖（金色下午光），冷色仅用于水面和夜晚章节。目标饱和度 55–75%，避免过度饱和的鲜艳卡通风。

AC × Glow Island 对照表

Animal Crossing 特征 Glow Island 对应体现 设计意图
柔和轮廓线，非纯黑描边 轮廓色 = 主色加深 30%（非 #000000） 避免视觉刺激，保持有机感
Q 版角色，大头圆身 Saki 和 NPC 头身比 1:1.5，眼睛占脸 1/3 提升亲和力与情感共鸣
木质 / 竹制 / 自然材料 UI 木牌对话框，树叶按钮，椰壳货币图标 强化日式离岛世界观沉浸感
每个岛民有独特剪影 每个 NPC 单色剪影可识别，无视分辨率 32px 缩略图仍可辨认身份
环境物件有轻微"呼吸"弹动 Idle 动画包含 0.5Hz 轻微上下浮动 赋予世界"活着"的生命感
柔和环境音映射视觉节奏 （音频参考，不在本文档范围） —

刻意与 AC 的差异化

  • 色调更偏向日式传统：增加鸟居红、藏青、竹绿
  • 游戏机制视觉（图块消除、连击反馈）用自然元素替代 AC 家具系统
  • 主角 Saki 有独立日式少女身份，非 AC 圆头动物造型

2.3 修复进度饱和度叙事

与 AC 岛屿"荒岛开拓"节奏相似，修复前是褪色速写，修复后是完整水彩画稿。

  • s0 修复前：饱和度 8–15%，亮度 42–48% — 褪色枯岛，铅笔草稿感
  • s1 第1阶段：饱和度 28–38% — 轮廓找回，淡彩涂色
  • s2 第2阶段：饱和度 48–58% — 生活气息，色彩入稿
  • s3 第3阶段：饱和度 62–72% — 角色回归，满生命力
  • s4 第4阶段：饱和度 75–88% — 完整水彩，光效全开

2.4 色彩对比度要求（降低认知负担核心规则）

规则参数目的
相邻章节图块色相差≥ 30°（HSL 色环）章节视觉分区明确
同章节内图块明度差≥ 20%快速识别图块类型
灰度模式可区分剪影原则，全部 20 种图块色觉障碍用户可用性
最低饱和度≥ 20%（禁止纯灰色图块）保持视觉活力

4.3 图块状态视觉规范

状态视觉效果技术方案
Normal 标准色，无特效，底部烘焙阴影 0 4px 0 rgba(0,0,0,0.10) 静态 PNG
Highlighted 白色内发光 rgba(255,255,255,0.4)，scale(1.06)，轮廓线变亮 Runtime Shader 或独立帧
Disappearing scale 1.0 → 1.12 → 0，旋转 ±12°，散出 4–6 个主题粒子（贝壳碎片 / 花瓣） Spine 序列帧，12 帧 @30fps
Special 持续呼吸发光（2.5s 周期），光晕颜色对应类型 Spine 循环动画
