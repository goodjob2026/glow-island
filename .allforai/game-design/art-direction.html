1. 整体风格定位

一句话定位

动物森友会式的温柔日式离岛：圆润、手绘、柔暖，随修复进度从褪色枯岛逐步"亮起"的治愈世界。

核心视觉哲学

为什么选择 Animal Crossing 风格
Animal Crossing 的视觉语言天然降低认知负担：每个物件轮廓简洁、颜色区分鲜明、信息密度低而情感密度高。玩家在直觉层面就能识别图块类型，不需要"读"图块——只需"感受"它。这与 Glow Island 的消除玩法高度契合：快速连接靠的是图形识别，而非文字逻辑。

三项核心原则
- 圆润优先：所有形状使用大圆角（≥16px），避免尖角。角色头身比接近1:1.5的Q版比例，四肢短而圆。
- 手绘质感：轮廓线保留轻微不规则感（模拟铅笔起稿），主色块用单色平铺+顶部极淡高光，整体如水彩图鉴插画风格。
- 暖色主导：整体色温偏暖（金色下午光），冷色仅用于水面和夜晚章节。避免过度饱和（目标饱和度55-75%，非鲜艳卡通风）。

与 Animal Crossing 的关键视觉共同点

AC特征 Glow Island 的对应体现
柔和轮廓线，非纯黑描边 轮廓色 = 主色加深30%（非#000000）
Q版角色，大头圆身 Saki和NPC头身比1:1.5，眼睛占脸1/3
木质/竹制/自然材料UI 木牌对话框，树叶按钮，椰壳货币图标
每个岛民有独特剪影 每个NPC单色剪影可识别，无视分辨率
环境物件有轻微"呼吸"弹动 Idle动画包含0.5Hz轻微上下浮动
柔和的环境音效映射视觉 （美术参考，不在本文档范围）

2.3 修复进度饱和度叙事

与AC岛屿"荒岛开拓"的节奏相似，修复前的岛屿是一张褪色速写，修复后是完整的水彩画稿。

修复前（s0）：HSL 饱和度 8-15%，亮度 42-48%  — 褪色枯岛，铅笔草稿感
第1阶段（s1）：饱和度 28-38%               — 轮廓找回，淡彩涂色
第2阶段（s2）：饱和度 48-58%               — 生活气息，色彩入稿
第3阶段（s3）：饱和度 62-72%               — 角色回归，满生命力
第4阶段（s4）：饱和度 75-88%               — 完整水彩，光效全开

2.4 色彩对比度要求（降低认知负担核心规则）

20种图块之间必须满足：
- 相邻章节图块色相差 ≥ 30°（HSL色环）
- 同章节内图块之间明度差 ≥ 20%
- 任意两种图块在灰度模式下仍可区分（剪影原则）
- 禁止使用纯灰色图块（最低饱和度 ≥ 20%）

3. 角色设计规范

3.1 主角 Saki（咲）的视觉设定

Saki 是岛上唯一的玩家角色，出现在关卡过渡动画和全岛地图界面中。

体型与比例
- 头身比：1:1.5（头部极大，接近AC村民比例）
- 头部：正圆形，直径占全身高40%；耳朵小圆形，略低于头顶
- 眼睛：水滴形，黑色瞳仁占眼面积70%，静止时略带向下弯曲的温柔感
- 鼻嘴：鼻子仅一个小圆点，嘴巴用简短弧线表达；AC式"省略鼻梁"原则
- 身体：圆筒形，无明显腰线；手臂短而圆，如两根小香肠
- 腿脚：短腿，脚部略大于手，增加稳定感和可爱感

服装（夏日离岛版）
- 上衣：宽松圆领短袖，主色 #F0C4C4 浅樱粉，袖口有细小竹叶印花（深绿#4A7D55线条，2px宽）
- 短裤：米白#F7EDD8，宽松百慕大款
- 帽子：宽沿草帽，本色麻纹，帽带为珊瑚色#E8866A
- 鞋：简单圆头凉鞋，棕色#C4955A
- 道具：永远背着一个海藻绿小背包，上面有贝壳形状扣子

表情规范（三套）
- Default（默认游览）：嘴角微微上翘，眼睛半开，放松状态
- Happy（关卡完成）：月亮形眯眼，嘴巴大"Ω"形，双手握拳举起
- Thinking（关卡提示）：一只眼眯起，嘴角歪向一侧，食指点嘴角

3.2 NPC 通用设计原则

所有NPC遵循与AC村民相同的"一眼识别"原则：

剪影规则
- 每个NPC有1个标志性道具/配件，使其黑色剪影与其他NPC完全不同
- 体型有明显差异：高/矮/圆/细，不允许两个NPC体型相近

颜色规则
- 每个NPC有1个"代表色"占服装主色，与其章节主色协调但不完全相同
- 皮肤色限定4个色值：#F5D4B0（浅）/ #E8B882（中）/ #C8935A（深）/ #9A6840（深褐）

面部规则
- 眼睛统一使用水滴形（同Saki）
- 允许添加：腮红（2个小圆点，主色降低明度40%）
- 允许添加：花纹/胡子/眼镜作为个性化标志

3.3 各章节 NPC 视觉细节

NPC01 — 渔夫·阿海（第1章 海边码头）
- 体型：中等身高，圆肩，手臂略粗（常年拉网）
- 代表色：靛蓝 #4A6D9A（布衣）
- 标志性配件：宽沿草帽（比Saki的更旧更大，帽沿有破损线条）+ 右手常拿鱼竿

- 腮红：无（沉稳气质）

NPC02 — 花农·阿花（第3章 樱花坡田）
- 体型：矮小圆润，头略大，圆脸

- 标志性配件：头顶别着一朵三瓣花（与章节T09相同花形），左手常端浇水壶
- 皮肤：浅 #F5D4B0
- 腮红：圆形深粉腮红，明显
- 特别规则：蹲地时身体变得更圆（膝盖消失，仅剩圆形轮廓）

NPC03 — 背包客·阿林（第4章 松林神社）
- 体型：偏高细，肩宽，有探险家气质
- 代表色：深绿探险外套 #3D6645

- 皮肤：中 #E8B882
- 腮红：无
- 特别规则：重要 — 标志性时刻（第4章完成）阿林抬头仰望星空，这是全章唯一露出眼睛的时刻，眼神设计为温柔上弯（AC"微笑眼"变体）

NPC04 — 温泉老爷爷·阿福（第5章 渔村港口）
- 体型：矮胖圆，肚子圆鼓（穿着缁色浴衣时如一颗圆球）

- 标志性配件：头顶始终顶着一块叠好的白色方巾（AC式夸张道具）
- 皮肤：中 #E8B882
- 腮红：深橙腮红，明显（笑纹配合腮红增强慈祥感）
- 特别规则：默认表情是"嫌弃/无奈"的眯眼撇嘴——眼睛向下弯（反向AC微笑眼），嘴角朝下，但腮红存在使其整体可爱而非凶狠

NPC05 — 灯塔守护者·阿明（第6章 夜晚灯塔）

- 皮肤：中 #E8B882
- 腮红：无
- 特别规则：标志性时刻（灯塔亮起）阿明背对玩家，仅看到大衣后背和点亮灯塔的剪影——这个构图本身就是情感高峰，无需正面表情

4. 图块视觉设计规范

4.1 图块基础规格

项目 规格
画布尺寸 @2x 128 × 128 px
实际绘制区 @2x 112 × 112 px（四周各留8px透明出血）
基础形状 圆角矩形（圆角半径 24px），如有自然形态则使用有机轮廓
轮廓线 2px，颜色 = 主色加深32%，非纯黑
底部阴影 0 4px 0 rgba(0,0,0,0.10) 烘焙进图层
纸纹叠加 6%不透明度的轻纸纹底，全部图块统一（模拟手绘纸感）

4.2 AC风格图块设计原则

有机形状优先：图块形状来源于自然物体（贝壳、叶片、花朵、鱼形），非几何图形。即使使用圆角矩形作底，也要在图块表面绘制清晰的自然主题插图。

剪影识别度：在32×32px大小下，每个图块的主题插图仍需可辨认。测试方法：将图块缩至25%大小，主题图案轮廓仍清晰。

色彩饱和度分布：图块主色饱和度60-75%，高光区饱和度降低15%，阴影区饱和度提升10%同时明度降低25%。

4.3 图块状态视觉规范

状态 视觉效果 技术方案
Normal（默认） 标准色，无特效，底部烘焙阴影 静态PNG
Highlighted（连接高亮） 白色内发光rgba(255,255,255,0.4)，scale(1.06)，轮廓线变亮 Runtime Shader（或独立帧）
Disappearing（消除） 先scale(1.12)再缩至0，出现3-5个对应主题的小粒子散开（贝壳块散出贝壳碎片，花朵块散出花瓣） Spine序列帧，12帧@30fps
Special（特殊块） 持续呼吸发光（2.5s周期），光晕颜色对应类型 Spine循环动画

图块类型 自然主题 视觉主色 特效描述

5. UI / UX 视觉语言

5.1 整体 UI 风格定义

木牌系 UI（Wood Sign UI）：所有UI元素的材质参考来自AC中的木质告示牌、竹简、晒干的贝壳和天然石板。

禁止使用的视觉元素
- 金属光泽（无铬钢、无铝合金边框）
- 硬边几何（无直角矩形，无菱形）
- 霓虹色（无荧光绿、电蓝、紫外色）
- 阴影过强（所有阴影透明度 ≤ 15%）

5.2 对话框 / 弹窗

  • 形状：圆角矩形，圆角半径 20px
  • 背景材质：米色木纹底（#E8D8B4）+ 顶部轻量手绘木纹线条（3-5条横向波浪纹，透明度15%）
  • 边框：4px 深棕木框线 #8B6030，外侧有1px米白高光 #F7EDD8（模拟木边缘）
  • 顶部装饰：对话框顶部中央有一枚竹钉图案（圆形，#7A5030，直径16px）
  • 文字：思源圆体 Regular，#3A2810深棕，行高1.6

5.3 按钮规范

主要按钮（确认/开始）
- 形状：宽度自适应，高度48px，圆角24px（药丸形）
- 背景：珊瑚橙渐变（顶 #F0997A → 底 #E8866A）
- 文字：月光白 #F5F0E8，思源圆体 Semi-Bold，16px
- 边框：2px 深珊瑚 #B85A40，底部有3px深珊瑚色偏移（AC式"按钮有重量感"）
- 点击态：整体下移2px（模拟物理按压），颜色整体加深8%

次要按钮（取消/关闭）
- 背景：米色木牌色 #E8D8B4
- 文字：深棕 #5A3820
- 边框：2px #B89060

图标按钮（道具类）
- 形状：56×56px 圆形
- 背景：米白#F7EDD8圆形底 + 主题色图标
- 边框：3px 棕色木纹圈 #C4955A
- 悬停/高亮：圆形变为主题色底色，图标变为白色

5.4 HUD元素规范

步数计数器
- 形状：叶形或扇贝形容器（非矩形）
- 背景：深珊瑚 #C85A40（低步数时变红 #C82020）
- 数字：月光白，Bold，28px，思源圆体数字字形
- 位置：屏幕右上角

分数/进度
- 形状：竹节形进度条（分段样式，每段为一节竹），圆角5px
- 填充色：竹叶绿 #7DBE77
- 底色：深棕木 #8B6030
- 段落之间用2px暗棕分隔线 #5A3820

连击泡泡（Combo Popup）
- 字体：思源圆体 Bold，连击数字极大（32-48px），颜色 #F5C842 金色
- 气泡形状：对话气泡轮廓，圆角矩形，底部有小三角尖（指向发生连击的区域）
- 背景：深珊瑚 #C85A40，边框月光白#F5F0E8 3px
- 入场动画：从下往上弹出，spring曲线，轻微overshooting（scale 1.2 → 1.0）

5.5 货币图标视觉

沙滩币（软货币）
- 形状：圆形，直径32px
- 图案：扇贝正视图，放射状细线，主色 #F5C842 金黄
- 边框：2px 深金 #B8902A
- 质感：表面有轻微贝壳光泽渐变

光石（硬货币）
- 形状：五边形宝石切面
- 图案：中央有星形内发光
- 主色：青绿渐变（#5BBFB5 → #3A9E94），高光为月光白
- 特效：静止时有0.8Hz轻微脉动光晕

5.6 字体规范

  • 中文主字体：思源圆体（Source Han Rounded CN）
  • 标题（关卡名）：Bold，22px，#3A2810
  • 正文（对话）：Regular，15px，#3A2810，行高1.65
  • 辅助信息：Light，12px，#7A6040
  • 数字：思源圆体（保持圆润感，与中文一致）
  • 步数/分数：Semi-Bold，20px
  • 连击数：Bold，32-48px（随连击层级增大）
  • 英文：系统圆体（SF Pro Rounded / Nunito），仅用于无中文替代的场景

5.7 图标风格

所有图标以AC中的物品手绘图鉴为参考：
- 风格：轻手绘线面结合，有轻微线条不规则感
- 禁止：锐利的几何矢量图标（无 Material Icon 风格）
- 推荐：图标内有一个主题自然元素（树叶/贝壳/花朵）作为设计基元
- 色彩：图标使用章节主色，单色原则（不超过2色）

6. 环境设计（6章区域）

6.1 整体环境风格原则

所有区域以AC的"俯视等距透视"为参考，但Glow Island棋盘区域为正视平面视角——环境设计主要体现在：
- 棋盘背景：可透视的渐变底图，透明度20-35%以不干扰图块识别
- 区域场景图（全岛地图视角）：45°俯视，手绘水彩风格
- 过渡动画背景：章节开始/结束时的全屏场景插画

7. 动画风格规范

7.1 AC 式动画原则

Animal Crossing的动画特征：弹性、呼吸感、物理质感。所有动画都像是实际重量的物体在互动，而非数字UI元素在切换。

四个关键词：慢起快落（ease-out） / 轻微过冲（slight overshoot） / 呼吸感（breathing） / 弹性（spring）

7.2 角色动画

Idle动画（所有NPC和Saki）
- 上身/头部轻微上下浮动：振幅2-3px，周期2.5-3s，ease-in-out
- 眼睛每4-6秒眨一次：眼皮从上方降下，动画0.15s
- 偶发动作（每8-12秒随机触发一次）：
- 左右轻摇头（旋转±5°）
- 手臂小幅摆动（仅前臂，±8°）
- 短暂"嗯？"抬头动作

Talk动画（对话中）
- 嘴巴开合循环：0.3s周期，嘴形在"闭合"和"半张"之间切换
- 身体轻微前倾：向玩家方向倾斜3-5°，配合说话节奏
- 眼睛轻微眯起（表达亲密感）：比idle状态眯10%

React动画（特定情绪）
- 开心：全身跳起（向上12px）再落下，弹性曲线，头顶出现星星/花瓣粒子（3-5个）
- 惊讶：头部向后仰，身体后退2px，眼睛变圆（瞬间放大），头顶出现"！"图案

7.3 图块动画

图块消除
- 帧1-3：图块scale从1.0→1.14（32ms）
- 帧4-7：图块scale从1.14→0，同时旋转±12°（随机方向）
- 同时散出：4-6个小粒子，形状为该图块的主题元素缩小版（贝壳块→小贝壳，花朵块→小花瓣），粒子运动方向随机，衰减消失（总时长0.3s）

连击气泡入场
- Spring曲线：stiffness=280, damping=22
- 从触发图块上方12px位置出现，向上浮动24px
- 出现时scale 0→1.2→1.0（带轻微过冲）
- 悬停1.2s后向上淡出（0.4s, ease-in）

图块Highlighted（连接状态）
- 进入高亮：scale 1.0→1.06，0.2s ease-out，同时内发光从0→目标亮度
- 退出高亮（连接取消）：scale 1.06→1.0，0.15s ease-in
- 保持高亮中：极慢速脉动，振幅±0.02 scale，周期1.5s

特殊块Idle呼吸
- Glow提灯块：内光芒scale 0.85→1.15→0.85，周期2.0s，ease-in-out，颜色随呼吸从#F5C842→#FFE580
- 彩虹鱼块：颜色条纹缓慢沿鱼身向尾方向流动，速度20px/s，不停歇
- 河豚块：轻微膨胀收缩，scale 0.95→1.05，周期2.8s，带轻微不规则性（非完美正弦）
- 金枪鱼块：身体轻微左右摆动（±3°），周期1.8s

7.4 UI动画

弹窗出现
- 背景遮罩：透明度0→0.5，0.3s，ease-out
- 弹窗本体：scale 0.7→1.05→1.0，0.4s，spring曲线

弹窗关闭
- 弹窗本体：scale 1.0→0.85，透明度1→0，0.25s，ease-in
- 背景遮罩：0.3s淡出

场景过渡（章节间）
- 白色圆形从中央向外扩散（AC换图过渡经典手法），0.5s cover + 0.5s reveal
- 全屏时显示下一章节的主题图标（贝壳/竹叶/樱花等）居中展示0.3s

修复完成动画
- 阶段切换：区域图片crossfade，0.8s ease-in-out
- 同步：白色光波从区域中心向外扩散（半径从0→全区域范围），透明度1→0，0.6s
- 第4阶段（满修复）：额外金色粒子雨，持续2.5s，粒子形状为该章节主题元素

8. 特殊视觉规则

8.1 禅境模式（Zen Mode）

  • 棋盘底图饱和度额外提升10%
  • 所有UI元素淡出至20%透明度（仅保留步数计数和棋盘本体）
  • 背景增加极慢速花瓣/树叶/泡泡粒子（速度0.3-0.8px/s，数量≤8个在屏幕内）
  • 图块消除时粒子效果更豪华（多出50%粒子量）

8.2 视觉降噪原则（认知负担控制清单）

  • 棋盘区域内，同屏内运动元素不超过3个（排除消除动画）
  • 所有背景装饰透明度 ≤ 15%，不与图块色相重叠
  • 特殊块持续动画不超过2种同时在棋盘中出现（程序控制刷新频率）
  • HUD元素总宽度 ≤ 棋盘宽度的20%（不侵占核心游戏区域）
  • 字体不超过2种字重混用于同一界面

8.3 刻意规避的视觉元素

规避项 原因
霓虹/赛博颜色（#00FFFF类） 与暖色日式调性完全冲突
金属高光（铬钢感渐变） 破坏手绘有机质感
像素艺术风格 目标用户审美期待更现代
过度粒子效果（屏幕碎片满天飞） 增加认知负担，破坏"轻盈"感
3D透视UI（拟物化极度写实） 与扁平有机风格冲突
纯黑色（#000000）使用 任何地方均替换为深棕#3A2810或深蓝#1E2E50
