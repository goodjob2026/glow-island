一、整体风格定位

核心定位

どうぶつの森式的温柔日式离岛：圆润、手绘、柔暖，随修复进度从褪色枯岛逐步"亮起"的治愈世界。
「温暖从不宣告，只是发生。」

视觉哲学：为什么选择 Animal Crossing 风格

Animal Crossing 的视觉语言天然降低认知负担：每个物件轮廓简洁、颜色区分鲜明、信息密度低而情感密度高。玩家在直觉层面即可识别图块类型，无需"读"图块——只需"感受"它。这与 Glow Island 的连连看消除玩法高度契合：快速连接靠的是图形识别而非文字逻辑。

三项核心原则

  • 圆润优先：所有形状使用大圆角（≥16px），避免尖角。角色头身比接近 1:1.5 的 Q版比例，四肢短而圆。
  • 手绘质感：轮廓线保留轻微不规则感（模拟铅笔起稿），主色块用单色平铺+顶部极淡高光，整体如水彩图鉴插画风格。
  • 暖色主导：整体色温偏暖（金色下午光），冷色仅用于水面和夜晚章节。饱和度目标 55–75%（非鲜艳卡通风）。

与 Animal Crossing 的差异化（避免直接复制）

  • 色调更偏向日式传统：增加鸟居红、藏青、竹绿，区别于 AC 鲜亮热带色调
  • 主角 山田律 为写实比例更接近的人类青年，非 AC 圆头动物造型
  • 游戏机制视觉（图块消除、连击反馈）以自然元素替代 AC 的家具系统元素
  • 叙事情感浓度更高：角色设计服务于「若有若无情感线」与「治愈而不催促」的调性

2.3 修复进度饱和度叙事

修复前的岛屿是一张褪色速写，修复后是完整的水彩画稿。饱和度随关卡进度线性提升，与岛屿视觉修复系统直接挂钩。

修复阶段 s0（修复前）：HSL 饱和度 8–15%，亮度 42–48%   — 褪色枯岛，铅笔草稿感
修复阶段 s1          ：饱和度 28–38%                    — 轮廓找回，淡彩涂色
修复阶段 s2          ：饱和度 48–58%                    — 生活气息，色彩入稿
修复阶段 s3          ：饱和度 62–72%                    — 角色回归，满生命力
修复阶段 s4（满修复）：饱和度 75–88%                    — 完整水彩，光效全开

2.4 色彩对比度规则（降低认知负担）

  • 相邻章节图块色相差 ≥ 30°（HSL色环）
  • 同章节内图块之间明度差 ≥ 20%
  • 任意两种图块在灰度模式下仍可区分（剪影原则）
  • 禁止使用纯灰色图块（最低饱和度 ≥ 20%）
  • 背景装饰透明度 ≤ 15%，不与图块色相重叠

三、角色设计规范

3.1 主角 山田律（やまだ りつ / Yamada Ritsu）

山田律 是玩家操控角色，出现在关卡过渡动画、剧情演出和全岛地图界面中。五年职场高压的上班族，来到 輝島（Kagayaki-jima）后逐渐解冻。

体型与比例

  • 头身比：1:1.5（Q版偏写实过渡，比 AC 村民略高挑，保留少量现实感）
  • 头部：接近正圆形，直径占全身高 38%；耳朵小圆形，略低于头顶
  • 眼睛：水滴形，黑色瞳仁占眼面积 65%，静止时略带内敛的疲惫感（与 NPC 的开朗眼神做区分）
  • 鼻嘴：鼻子仅一个小圆点，嘴巴用简短弧线；AC 式"省略鼻梁"原则
  • 身体：圆筒形，无明显腰线；手臂短而圆；腿脚：短腿，脚部略大于手

服装（夏日离岛版）

  • 上衣：宽松圆领短袖，主色 #E0E8D8 浅灰绿（日常感，非鲜艳）
  • 外搭：浅米色开衫，随剧情进度增减（Ch4 营地时脱下，放在石头上）
  • 短裤：深灰蓝 #3A4A60，直筒款
  • 帽子：浅灰软沿帽（非草帽）——区别于 AC 草帽形象
  • 鞋：简单圆头休闲鞋，深棕 #5A3820
  • 标志性道具：随身带一本略皱的笔记本（用于写日记），封面灰绿色

表情规范（三套）

表情名称适用场景视觉描述
默认（待机）关卡间隙、地图浏览嘴角平直略带轻松，眼睛半开，肩膀放松（与初始紧绷形成对比）
开心（关卡完成）通关结算、剧情触发眼角上弯，嘴巴 Ω 形微开，略带意外感——不是狂喜，是"哦，还不错"的惊喜
思考（关卡提示）步数不足时触发一只眼眯起，嘴角歪向一侧，食指轻点嘴角，视线偏向棋盘

3.2 NPC 通用设计原则

所有 NPC 遵循"一眼识别"原则：黑色剪影无需颜色即可区分；每位 NPC 有独一无二的标志性道具/配件。

剪影规则

  • 每个 NPC 有 1 个标志性道具/配件，使其黑色剪影与其他 NPC 完全不同
  • 体型有明显差异：高/矮/圆/细，禁止两个 NPC 体型相近

颜色规则

  • 每个 NPC 有 1 个"代表色"占服装主色，与其章节主色协调但不完全相同
  • 皮肤色限定 4 个色值：#F5D4B0（浅）/ #E8B882（中）/ #C8935A（深）/ #9A6840（深褐）

面部规则

  • 眼睛统一使用水滴形（同山田律）
  • 允许添加：腮红（2个小圆点，主色降低明度 40%）
  • 允许添加：花纹/胡子/眼镜作为个性化标志

3.3 各章节 NPC 视觉细节

第1章 · 海边码头 · Ch1向导

磯部健三（いそべ けんぞう / Isobe Kenzo）— 老渔夫

  • 体型：中等身高，圆肩，手臂略粗（常年拉网），后背微微弓起
  • 代表色：#4A6D9A 靛蓝布衣（做旧感，不鲜亮）
  • 标志性配件：宽沿草帽（帽沿比一般大且有破损线条）+ 右手常拿鱼竿或打捞物
  • 皮肤：#9A6840 深褐（常年日晒）
  • 腮红：无（沉稳气质）
  • 设计要点：Ch1 多数时候背对玩家看海，正面亮相仅在标志性情感时刻。动画中记录册道具可见于腰间。
第2章 · 中央小镇 · Ch2

桐島梅子（きりしま うめこ / Kirishima Umeko）— 陶艺工匠

  • 体型：矮小圆润，头略大，圆脸，手掌略宽（揉陶土的质感）
  • 代表色：#7DBE77 草绿围裙 + #F0C4C4 浅粉底裙（碎花细印）
  • 标志性配件：围裙上常有陶土痕迹（不规则灰棕点状）；双手偶有淡灰泥痕
  • 皮肤：#F5D4B0 浅
  • 腮红：圆形深粉腮红，明显（眼力精准却面容亲切）
  • 设计要点：工作室最显眼处始终放着律的脸碗（浮雕，右眉小疤）。蹲地时身体变得更圆，膝盖消失仅剩圆形轮廓。
第3章 · 花田山坡 · Ch3

海野夏帆（うみの なほ / Umino Naho）— 自然向导

  • 体型：偏高细，轻盈感，常蹲下观察植物（蹲姿是标志性姿态）
  • 代表色：#3D6645 深绿探险上衣 + 米色宽松长裤
  • 标志性配件：渔夫帽帽沿低压（多数时候遮住眼睛，仅露嘴巴与下巴），腰间挂着小植物标牌
  • 皮肤：#E8B882 中
  • 腮红：无，但耳尖偶有淡粉（害羞说出真心话时）

第4章 · 森林露营地 · 独处章节（无主 NPC，但ひなた痕迹在场）

本章角色缺席说明

第4章 山田律 独处，无主要 NPC 出场。场景中保留 ひなた 留下的符号木板（贴于山路岔路口）、野花标记，以及远处冬子厨房的灯光。议长（橘猫）在营地边绕行出现。

  • ひなた 留下物的视觉：小木板，白桦色，表面有手写符号（自创符号体系，旁人看不懂），下压一束半干野花，花色为 #F0C4C4 浅樱粉

第5章 · 温泉山谷 · 核心章节（恋爱线最深）

丸山ひなた（まるやま ひなた / Maruyama Hinata）— 灯台守孙女

  • 体型：接近律的比例，略矮半头，动作习惯轻柔——放东西时总是先确认不会打扰到什么，再轻轻放下
  • 代表色：#F0C4C4 浅樱粉圆领短袖 + 浅米白裙 + 珊瑚色帽带草帽
  • 标志性配件：随身带一本边角起毛的手账（符号体系记录本），封面深蓝布面，总夹在手肘或腋下
  • 皮肤：#F5D4B0 浅
  • 腮红：淡圆粉腮红，低调（不过分可爱化）
  • 气味视觉化：律认出 ひなた 靠气味（海水+碳粉墨水）。过场动画可用极淡的蓝灰色墨水粒子散在她指尖周围，仅在亲密感提升的演出时刻出现，透明度 20%，不过分显眼。
  • 设计要点：Ch5 标志性时刻 —— 律第一次钓到鱼，ひなた 侧过脸说「还行」，两字分量因前面对话而完全不同。表情：嘴角极轻上扬，眼神平和略带内容，非大笑。
第6章 · 夜晚灯塔 · Ch6

丸山冬子（まるやま ふゆこ / Maruyama Fuyuko）— 灯台守遗孀

  • 体型：偏高，微弯腰（习惯性的等待姿态），步伐缓慢而坚定
  • 代表色：#2A4060 藏青外罩衫 + 浅灰内搭，腰带为 #C4955A 棕色布带
  • 标志性配件：左手偶有一个小木箱（存放别人暂时拿不动的东西）；灯塔旁有一把折叠椅是她常坐的地方
  • 皮肤：#E8B882 中（岁月感，但非老态）
  • 腮红：无，眼角有轻微细纹（设计允许）
  • 设计要点：点灯仪式时，冬子在塔底，背对观看方向，望着灯塔说出「渉さん、見えてますよ。」这个构图本身是情感高峰——无需正面表情，背影剪影即足够。
贯穿全章 · 全岛

议长（橘猫 / Gichou）— 岛上最老的居民

  • 体型：圆润肥短的橘猫，四肢极短，尾巴微翘，坐下时是一个近乎正圆的形态
  • 配色：主体橘色 #E8A050，额头白色宽纹，腹部白，尾尖白，眼睛琥珀黄 #D4900A
  • 标志性姿态：坐着不动；离开时永远向左走。健三的记录册专门有一栏记录"裁决方向"。
  • 设计要点：议长在旁边，总觉得事情没那么难。他的存在本身是一种最不占地方的陪伴。每晚陪冬子在灯台旁坐着，是唯一固定日程。律到岛第三天，议长坐在律脚边 12 秒后向左离开——这是岛对律最轻的接纳仪式。

最小识别尺寸缩至 32×32px 时，主题图案轮廓仍清晰可辨

4.2 图块设计原则

  • 有机形状优先：图块形状来源于自然物体（贝壳/叶片/花朵/鱼形），非几何图形。即使使用圆角矩形作底，也要在表面绘制清晰的自然主题插图。
  • 剪影识别度：在 32×32px 大小下，每个图块的主题插图仍需可辨认。
  • 色彩饱和度分布：图块主色饱和度 60–75%，高光区饱和度降低 15%，阴影区饱和度提升 10% 同时明度降低 25%。
  • 禁止跨图块复用色相（同章节内）：相邻图块色相差 ≥ 30°。

六、环境设计（六章区域）

6.1 整体环境风格原则

  • 棋盘背景：可透视的渐变底图，透明度 20–35%，不干扰图块识别
  • 区域场景图：45° 俯视，手绘水彩风格（全岛地图视角）
  • 过渡动画背景：章节开始/结束时的全屏场景插画
  • 色彩引导情感：饱和度随修复阶段上升，视觉苏醒先于对话
  • 行动优于台词：健三递茶；梅子把碗放在最显眼处；冬子不敲门放下食物就走
  • 灯塔作为情感锚：每次修复，灯塔窗里的光多一点点

第1章 — 海边码头（海边の波止場）

情绪基调：悠闲的南海小渔村午后，初抵褪色之岛

  • 修复后关键元素：弯曲木码头 #C4955A 带轻微木纹；两侧竹制灯笼（圆形，#EE8C55橙光）；停靠小木船（船头挂贝壳风铃）；告示板上手写便条「欢迎来到輝島，请随便走走」无署名

第2章 — 中央小镇（中央の町）

  • 修复后关键元素：石板路清扫（梅子工作室的门能完全打开）；喷泉修复（第一次出水时水柱歪了一下，然后直了）；店铺门牌修缮（字迹还是老式的，没有换成新的）；小镇中央座椅修缮（健三第一个来坐）

第3章 — 花田山坡（花畑の丘）

  • 修复后关键元素：花田小径清理（夏帆说路好走多了）；瞭望台修复（可以看见灯塔了，虽然灯还是灭的）；野花圃重整（夏帆认真告诉每株花「工程结束了，谢谢配合」）；山坡石阶修缮（ひなた带律走完每一阶）

第4章 — 森林露营地（森のキャンプ地）

  • 修复前：帐篷架生锈倒塌，篝火台积灰，营地小径木牌褪色

第5章 — 温泉山谷（温泉の谷）

七、动画规范

7.1 动画核心原则（Animal Crossing 式）

Animal Crossing 动画特征：弹性、呼吸感、物理质感。所有动画都像是实际重量的物体在互动，而非数字 UI 元素在切换。

四个关键词：慢起快落（ease-out）/ 轻微过冲（slight overshoot）/ 呼吸感（breathing）/ 弹性（spring）

7.2 角色动画规范

待机动画（所有 NPC 和山田律）

  • 上身/头部轻微上下浮动：振幅 2–3px，周期 2.5–3s，ease-in-out
  • 眼睛每 4–6 秒眨一次：眼皮从上方降下，动画 0.15s
  • 偶发动作（每 8–12 秒随机触发一次）：左右轻摇头（旋转 ±5°）/ 手臂小幅摆动（仅前臂，±8°）/ 短暂"嗯？"抬头动作

对话动画

  • 嘴巴开合循环：0.3s 周期，嘴形在"闭合"和"半张"之间切换
  • 身体轻微前倾：向玩家方向倾斜 3–5°，配合说话节奏
  • 眼睛轻微眯起（亲密感）：比待机状态眯 10%

反应动画

  • 开心：全身跳起（向上 12px）再落下，弹性曲线，头顶出现星星/花瓣粒子（3–5 个）
  • 惊讶：头部向后仰，身体后退 2px，眼睛变圆（瞬间放大），头顶出现"！"图案

DragonBones 骨骼动画目标列表

  • 山田律 — 全身骨骼动画 idle/talk/react/篝火夜仰星动作
  • 磯部健三 — 骨骼动画含标志性背对码头系绳时刻、记录册翻写动作
  • 桐島梅子 — 骨骼动画含蹲地圆形轮廓变形、揉陶土手部细节
  • 海野夏帆 — 骨骼动画含蹲下观察植物姿态、递花束动作
  • 丸山冬子 — 骨骼动画含灯塔点亮背影剪影、放食物于门口不敲门动作
  • 丸山ひなた — 骨骼动画恋爱兴趣角色，含手账翻写、钓鱼台旁「还行」侧脸
  • 议长（橘猫）— 骨骼动画含坐定/向左离去/抬头看灯塔光

7.3 图块动画规范

图块消除

  • 帧 1–3：图块 scale 从 1.0 → 1.14（32ms）
  • 帧 4–7：图块 scale 从 1.14 → 0，同时旋转 ±12°（随机方向）
  • 同时散出：4–6 个小粒子，形状为该图块的主题元素缩小版（贝壳块散出小贝壳，花朵块散出小花瓣），粒子运动方向随机，衰减消失（总时长 0.3s）

连击气泡入场

  • Spring 曲线：stiffness=280, damping=22
  • 从触发图块上方 12px 位置出现，向上浮动 24px
  • 出现时 scale 0 → 1.2 → 1.0（带轻微过冲）
  • 悬停 1.2s 后向上淡出（0.4s, ease-in）

图块高亮（连接状态）

  • 进入高亮：scale 1.0 → 1.06，0.2s ease-out，内发光从 0 → 目标亮度
  • 退出高亮：scale 1.06 → 1.0，0.15s ease-in
  • 保持高亮中：极慢速脉动，振幅 ±0.02 scale，周期 1.5s

特殊块待机呼吸

  • 萤火虫提灯块：内光芒 scale 0.85 → 1.15 → 0.85，周期 2.0s，颜色从 #F5C842 → #FFE580
  • 彩虹鱼块：颜色条纹沿鱼身向尾方向缓慢流动，速度 20px/s，不停歇
  • 河豚块：轻微膨胀收缩，scale 0.95 → 1.05，周期 2.8s，带轻微不规则性
  • 金枪鱼块：身体轻微左右摆动（±3°），周期 1.8s

动画类型参数

八、特殊视觉规则

8.1 视觉降噪原则（认知负担控制清单）

  • 棋盘区域内，同屏运动元素不超过 3 个（排除消除动画）
  • 所有背景装饰透明度 ≤ 15%，不与图块色相重叠
  • 特殊块持续动画不超过 2 种同时在棋盘中出现（程序控制刷新频率）
  • HUD 元素总宽度 ≤ 棋盘宽度的 20%（不侵占核心游戏区域）
  • 字体不超过 2 种字重混用于同一界面
  • 特殊块动画不遮挡棋盘连接路径（VFX 层低于 HUD 层，高于背景层）

8.2 刻意规避的视觉元素

规避项原因
霓虹/赛博颜色（#00FFFF 类）与暖色日式调性完全冲突
金属高光（铬钢感渐变）破坏手绘有机质感
像素艺术风格目标受众审美期待更现代
过度粒子效果（屏幕碎片满天飞）增加认知负担，破坏"轻盈"感
3D透视UI（拟物化极度写实）与扁平有机风格冲突
纯黑色 #000000任何地方均替换为深棕 #3A2810 或深蓝 #1E2E50
竞技感、紧迫感视觉（警报红、闪烁提示）与治愈调性冲突；低步数警示仅用颜色变化，无闪烁
强制视觉弹窗打断（侵入式购买提示）破坏沉浸感；付费引导轻量化

8.3 叙事视觉特殊规则

  • 修复饱和度叙事：s0 褪色草稿→s4 完整水彩，视觉上的苏醒先于对话，颜色本身在说故事
  • 灯塔作为全局情感锚：每个章节的瞭望点都能看见灯塔，灯塔灯室随章节进度亮度微微提升
  • 声音先于视觉：晨风里的远处风铃声预示修复，早于场景更新（音频设计节点对应）
  • 行动优于台词：健三递茶/梅子把碗放在最显眼处/冬子不敲门放食物——这些场景的视觉演出不依赖大量对话文字，靠动作叙事
  • 石刻谜题视觉一致性：三块风化石刻（Ch1码头/Ch3山坡/Ch5灯台基石）字体和风化程度需完全一致，均采用手刻刀痕风格，深灰 #6A6A6A 刻槽，石板底色与所在章节环境色协调
