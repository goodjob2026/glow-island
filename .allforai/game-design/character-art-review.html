动画ID 游戏状态 时长(ms) 循环 优先级 制作模式 关键姿势

root
  └─ pelvis
       ├─ torso（含neck合并）
       │    ├─ head（含hair_front/hair_back）
       │    ├─ upper_arm_l → forearm_l → hand_l
       │    └─ upper_arm_r → forearm_r → hand_r
       ├─ upper_leg_l → lower_leg_l → foot_l
       └─ upper_leg_r → lower_leg_r → foot_r
