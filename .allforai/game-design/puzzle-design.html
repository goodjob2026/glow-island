1. 路径算法 — BFS ≤2转弯规则，传送门0转弯穿越

两图块可连接的条件：类型相同，且存在至少一条路径满足以下四项约束：

转弯次数 ≤ 2 路径不穿过已占用图块 空格可穿越 可沿棋盘外边缘延伸 传送门零代价折叠

BFS 状态机

每个搜索状态由四元组 (x, y, last_direction, turns_used) 描述。

初始状态：将起始图块坐标以 (x₀, y₀, NONE, 0) 入队
↓ 扩展四个方向 (UP / DOWN / LEFT / RIGHT)
↓ 方向未改变 → turns_used 不变
↓ 方向改变 → turns_used += 1
↓ turns_used > 2 → 剪枝，不入队
↓ 遇到 portal_entry → 以相同 (direction, turns_used) 将 portal_exit 坐标入队，不增加转弯计数
↓ 到达目标图块坐标 → 路径有效，连接成功
↓ 队列耗尽未到达目标 → 不可连接

转弯计数可视示例

路径形状转弯次数是否有效说明
直线0有效同行或同列直接相连
L形1有效一次拐弯
Z/S形 / U形2有效两次拐弯，最常见路径
三折路径3无效超出上限，BFS剪枝
传送门直线穿越0有效门间折叠不计转弯
传送门 + 1次拐弯1有效传送门等于免费折叠点

传送门穿越规则

传送门对：棋盘上成对出现（同色标记），每组2个，关卡可有多组。
路径进入 portal_entry 格子时，BFS 将 portal_exit 坐标以相同方向和相同 turns_used 直接入队。
实现提示：BFS 遍历前先构建 portal_map: coord → paired_coord；遇到 portal 格子时查表替换，无需额外分支处理。

3. 8种障碍系统 — 机制、消除规则、视觉提示

共8种障碍按章节逐步引入。现有3种 v2新增5种

Ch2 解锁 冰封块（Ice Block）
ice_block · 1–3 层
机制
图块被冰层封印，无法被玩家选中或参与连接。支持 1–3 层叠加，每次相邻格子的消除操作击碎一层。前 N-1 次消除图块本身不消除，第 N 次消除后冰层完全破碎，图块正常消除。
消除规则
相邻消除（adjacent_match），每次相邻格子成功匹配消耗一层冰；总击中次数 = 层数。路径可绕过冰封块，但无法穿越已占用的冰封图块本体。
视觉提示
Layer 3: 厚重冰柱 + 白色反光；Layer 2: 龟裂纹路；Layer 1: 薄冰残片。最后一层破碎时：冰晶炸裂 + 水滴粒子飞溅，配合"咔嚓"碎冰音效。
Ch2 解锁 杂草（Weed）
weed · 动态扩散
机制
杂草占据格子，覆盖的格子无法放置图块（消除后格子恢复可用）。每 15秒向相邻1格扩散，不阻挡路径。
消除规则
相邻消除1次即可清除单格杂草。
视觉提示
绿色草丛覆盖格子。消除时叶片飞散动画。
Ch3 解锁 锁链（Chain Lock）
chain_lock · 1–3 环
机制
图块被锁链捆绑，必须将该图块本身匹配 N 次（每次直接参与连接算一次）才能解除锁链。前 N-1 次匹配图块不消除，仅减少锁链环数；第 N 次匹配后锁链脱落，图块正常消除。
消除规则
直接匹配（direct_match），图块本身必须参与 N 次成功匹配，相邻消除不计数。图块在锁链解除前可被选中。
视觉提示
金属锁链缠绕图块外框，每次匹配一个链环断裂 + 金属叮当音效。最后一环断裂时锁链飞散 + 图块发光消除。
Ch3 解锁 传送门（Portal）
portal · 路径工具，成对出现
机制
棋盘上成对出现的传送门格子（同色配对）。连线路径进入传送门入口，从配对出口继续延伸，传送过程 不消耗转弯次数（turn_cost = 0）。传送门格子本身不放置普通图块，仅作路径通道。
消除规则
传送门不可消除，属于永久地形元素，由关卡JSON定义。每关可有多对传送门（不同颜色区分配对）。
视觉提示
两个旋转发光圆环（同色），入口有吸入粒子旋涡。路径穿越时：粒子从入口吸入、从出口喷出，配合传送"嗖"声音效。

v2.0 新增5种障碍

v2新增 · Ch4 解锁 单路径通道（Single-Path Corridor）
single_path_corridor · 路径约束
机制
棋盘特定区域被设定为单路径通道，该区域内只有一条有效路径可通过。所有经过该区域的连线必须通过强制通道，不可绕行。通道格子本身不增加转弯计数，但进出通道的方向变换照常计入转弯。通道区域两侧格子被标记为不可穿越，迫使路径沿通道方向行进。
消除规则
不可消除，属于永久路径约束地形，由关卡JSON中的 corridor 坐标列表及强制通行方向字段定义。
视觉提示
通道格子有发光箭头指示方向，边界有虚线围栏。路径经过时通道格子依次点亮，配合轻微引导提示音。
v2新增 · Ch4 解锁 木箱（Wooden Crate）
wooden_crate · 永久地形障碍
机制
永久性地形障碍，无法消除。路径不可穿越木箱格子，图块不可放置于木箱格子上。用于关卡设计师强制规划路径走向，创造高难度谜题空间。与单路径通道组合可设计极高难度关卡。
消除规则
永久障碍，clear_method = none。不响应任何消除操作。
视觉提示
实木纹理箱体，边缘有金属扣。不可交互时显示禁止图标，鼠标悬停有抖动拒绝动画。
v2新增 · Ch5 解锁 水流（Water Current）
water_current · 动态路径障碍
机制
无法消除的动态障碍。每5秒改变流向，影响该格路径可达性。流向改变时实时更新路径计算——流向可能开放或封闭特定路径，要求玩家把握时机。
消除规则
不可消除，属于永久动态地形。BFS需在每次流向变化后重新计算当前可达性。
视觉提示
流动水波纹理格子，箭头图标指示当前流向。方向切换时有旋转动画，并伴随流水声音效变化。
v2新增 · Ch6 解锁 扩散障碍（Spreading Obstacle）
spreading_obstacle · 藤蔓 / 苔藓
机制
藤蔓或苔藓从初始格子向外扩散，被覆盖的图块无法被选中参与连接。每回合向上下左右各1格扩散（若未被占用）。若扩散填满整个棋盘则关卡失败。占据80%可用格子时触发视觉警告。扩散被 wooden_crate、portal、single_path_corridor 边界阻挡。
消除规则
相邻消除 1次即可清除单格覆盖。初始格子由关卡JSON指定（通常2–4个），扩散间隔 = 每步1格（步数制关卡）。
视觉提示
从初始格子向外蔓延的绿色藤蔓纹理（vine）或暗绿苔藓（moss）。每次扩散触发时有嫩芽破土动画；相邻消除清除时藤蔓枯萎收缩 + 叶片飞散；接近满盘时整体颜色变深红警告。

4. 特殊块详解 — 4种触发时机/效果/连锁

所有特殊块激活动画不可跳过（0.3–0.8s），以保持节奏感。特殊块激活可与障碍清除形成连锁反应。

炸弹（Bomb）
id: bomb · 动画 0.5s
触发时机
3连击时系统必定自动生成1个，或由关卡JSON预置于棋盘（默认5%空格随机概率）。
效果
消除以炸弹块为中心的 3×3 范围内所有图块（含障碍物及覆盖层）。范围内的冰封块、锁链、扩散障碍均被一并清除。
连锁规则
若3×3范围内有其他特殊块，被消除的特殊块不会二次激活（防止无限连锁）。被清除的障碍物正常计分。
视觉提示
橙色闪光球体。激活时：爆炸粒子 + 冲击波环向外扩散，3×3范围内图块逐格飞散。低频"砰"爆炸音效。
风车（Windmill）
id: windmill · 动画 0.4s
触发时机
4连击时系统必定自动生成1个，或由关卡JSON预置（默认3%空格概率）。
效果
同时清除风车块所在整行（横）及整列（竖）的所有图块，形成十字清除（行消和列消同步触发）。
连锁规则
十字范围内遇到冰封块/锁链均可清除。若十字范围内有炸弹块，炸弹块被清除但不触发炸弹爆炸。
视觉提示
旋转风车纹理。激活时：横竖双光柱同时扫过整行整列，图块向两侧飞散。高频"唰唰"扫除音效。
灯光（Lantern）
id: lantern · 动画 0.3s
触发时机
仅关卡JSON预置（不可通过连击生成）。棋盘上存在时自动激活。若无同类型图块可达，灯光块持续脉冲等待。
效果
自动寻路并连接距离最近的同类型图块，无需玩家手动操作。连接成功后双方同时消除。搜索范围无上限，使用全棋盘BFS。
连锁规则
自动连线不受障碍物影响（直接点对点光线连接，跳过障碍）。消除后若新出现的最近同类型图块仍满足条件，灯光块不会再次激活（单次触发）。
视觉提示
橙色光晕持续脉冲。激活时：发光连线高速飞向目标，目标图块同时发光响应，双方爆发光点消除。
海浪（Wave）
id: wave · 动画 0.8s
触发时机
仅关卡JSON预置（不可通过连击生成），通常每关最多1个。玩家主动匹配海浪块时激活。
效果
重新随机排列棋盘上所有图块位置，保持各类型图块数量不变。重排后若仍无可用匹配对则自动再次重排，最多3次。
连锁规则
重排不改变障碍物位置（木箱、传送门等固定地形保持原位）。重排后立即重新计算全盘可达性。重排期间不可进行任何操作。
视觉提示
蓝色水波纹从中心扩散至边缘。激活时：全棋盘图块漩涡式上扬后重新落下，配合"哗"海浪音效，持续0.8s。

6. 棋盘事件 — tile_fall / tile_slide / board_rotate 机制

棋盘事件均在第5章解锁，默认关闭，由关卡JSON中的 board_event 字段按需启用。

方块下落（Tile Fall）
tile_fall
Ch5 解锁 · 重力系统核心
触发：每次消除操作完成后立即触发，优先级最高。
机制：消除图块后，被消除格上方的图块因重力向下滑落填充空位；若顶部有补充池则从顶部补入新图块。
缓动：ease_in_bounce，每格下落耗时 0.08s，强化物理真实感。
连锁：下落到位后若形成新的可消除对，系统提示（不自动消除，等待玩家操作）。
约束：需在 6×6 以上棋盘使用；与 tile_slide 配合使用效果最佳。
方块滑落（Tile Slide）
tile_slide
Ch5 解锁 · 需配合 tile_fall
触发：tile_fall 完成后，若图块落在标记为"滑面"的格子上则触发水平滑动。
机制：特定标记为 slide_surface 的格子上，图块在下落后额外向指定水平方向滑动1格，形成斜向堆积效果。
缓动：水平滑动使用 ease_out，持续 0.12s。
配置：由关卡JSON标记 slide_surface 格子及滑动方向（left / right）。
设计用途：可创造图块聚集特定区域的战术设计，配合连击策略。
棋盘旋转（Board Rotate）
board_rotate
Ch5 解锁 · 仅挑战关使用
触发：每 N 步操作后触发（N 由关卡JSON指定，典型值 3–5 步）。
机制：棋盘特定矩形子区域（由关卡JSON定义）每 N 步后顺时针旋转 90°，改变图块位置布局，要求玩家预判旋转后状态。
预警：旋转前 1 秒在旋转区域边框闪烁警告，给玩家预判时间。
缓动：ease_in_out，旋转动画 0.5s。
约束：旋转区域不可包含 portal 或 single_path_corridor 格子；旋转完成后重新计算全盘可达性。仅用于章节后半段高难度挑战关。

7. 天天爱消除爽快感设计原则

参考手感：天天爱消除（Chinese Candy Crush）。每次成功匹配必须有清晰、爽快的视觉+音效反馈，连击链产生渐进增强的爆炸/粒子特效，3连击以上全屏震动。

  • 即时清晰反馈：每次成功匹配必须有清晰、爽快的视觉+音效反馈——消除瞬间粒子飞散、连击计数跳动、专属音效播放，响应延迟不超过 16ms。
  • 渐进增强连击：连击链应产生渐进式增强的爆炸/粒子特效；3连击以上全屏微震，4连击以上全屏震动 + 粒子爆发，让玩家的节奏感和成就感同步上升。
  • 不可跳过的精彩时刻：特殊块激活时动画不可跳过，持续时间控制在 0.3–0.8s，以保持节奏感——短到不打断流程，长到足以令人印象深刻。
  • 障碍解锁专属特效：障碍物被清除时应有专属"解锁"音效 + 碎裂/解冻/断链粒子效果，让"攻克障碍"成为独立的爽快时刻。
  • 弹性重力下落：重力下落使用 ease_in_bounce 缓动，强化物理真实感，让棋盘有"重量"和"活力"。
  • 新机制引导零压力：每章首次引入新机制时，通过引导关卡（无步数惩罚）让玩家自然学会——机制介绍用视觉引导而非文字墙，确保玩家第一次接触时产生"原来如此"而非困惑的情绪。
