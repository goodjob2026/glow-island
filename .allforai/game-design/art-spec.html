2 · 图块资产表

基础规格

画布尺寸 @2x128 × 128 px
实际绘制区112 × 112 px（四周8px出血）
颜色深度RGBA 32-bit
导出格式PNG-24（透明背景）
图集打包Texture Atlas，≤ 2048×2048
轮廓线2px，主色加深32%（非纯黑）
底部烘焙阴影0 4px rgba(0,0,0,0.10)
纸纹叠加6% 不透明度统一纹理

四种状态规范

状态 文件命名模式 帧数 说明
Normal {id}_{name}_normal@2x.png 1帧 静态，烘焙阴影 + 纸纹
Highlighted {id}_{name}_highlighted@2x.png 1帧（或Shader） 亮度+20%，内发光 rgba(255,255,255,0.40)
Disappearing {id}_{name}_disappear_{01-12}@2x.png 12帧 @30fps（0.4s） 帧1-3放大→帧4-9旋转缩小→帧10-12主题粒子散开
Special {type}_idle.skel / {type}_activate.skel Spine循环+一次性 仅4种特殊块，idle周期2.0-2.8s

4 · 场景资产

背景基础规格

尺寸1080 × 1920 px（Portrait 9:16）
安全区中央1080×1600，顶底各240px出血
颜色深度RGB 24-bit（无透明通道）
导出格式PNG-24 或 JPEG Q95
分层可选天空层 / 中景层 / 近景层（视差）
风格手绘水彩插画，AC式正视平面

修复前/后说明

版本后缀色调关键视觉
修复前 _before HSL饱和度 ≤ 15%，偏灰暗 破败但轮廓完整，如铅笔速写稿
修复后 _after 满彩水彩质感，明亮温暖 章节特色景观完整，角色出现

6 · 特效资产

特效基础规格

帧率30fps（大多数） / 60fps（核心）
格式PNG序列帧 + Spine骨骼动画
透明通道RGBA，全部透明背景
粒子风格自然元素（花瓣/气泡/竹叶），非抽象几何
EF01
图块消除特效（Tile Disappear）
画布：160×160 @2x   帧数：12帧 @30fps（0.4s）  触发：任意图块消除
格式：PNG序列帧  命名：fx_tile_disappear_{color}_f{01-12}@2x.png
6色系：teal / pink / gold / green / red / navy
粒子对应：T01→贝壳碎片 · T03→水滴 · T05→竹叶 · T09→花瓣 · T15→红色碎片 · T19→星点

画布：220×110 @2x   格式：Spine fx_combo_popup.skel
触发：连击 ≥ 3 时弹出   动画：spring弹入→悬停1.2s→上浮淡出（总~1.9s）
样式：深珊瑚背景 #C85A40，月光白边框3px，数字金黄 #F5C842
变体：3×3 / 5×5 / 10×10（字号 24px / 32px / 48px 递增）

7 · 交付规范

通用文件要求

分辨率
@2x 为主交付，@1x 通过自动缩小生成
色彩空间
sRGB
PNG 文件大小
单张 ≤ 500KB，大背景 ≤ 1.5MB
Spine 纹理大小
≤ 512KB
透明通道
边缘无多余像素，无白边/黑边溢出
命名规则
全部小写+下划线，含@2x/@1x后缀，不含中文/空格
版本号
更新版本使用 _v1 / _v2 后缀
风格一致性
全部图块在灰度图中，明度排序符合对比度要求

Spine 动画交付要求

Spine 版本
4.1.x（与引擎版本对齐）
导出格式
Binary .skel + JSON .json（备用）+ .atlas + PNG
纹理尺寸
2的幂次：256 / 512 / 1024
动画命名
全英文小写+下划线，如 idle / activate / talk / react_happy
Slots
可替换颜色/内容部分使用Slot，便于程序化控制
