1. 核心循环（Core Loop）

每局关卡时长 30 秒 – 3 分钟，以步数制驱动，无倒计时压力。

选关 → 连连看（30s–3min） → 材料奖励 → 岛屿修复进度 → 章节解锁
选关  →  进入关卡（律翻出旧物 / 走进荒芜角落）
↓ 观察棋盘布局（图块颜色 / 位置 / 特殊块分布）
↓ 规划连接路径（同色记忆碎片连线，路径可弯折，最短2个）
↓ 触发消除 → 即时反馈（音效 + 粒子特效 + 连击计数）
↓ 连击积累 → 触发特殊块生成 / 光能加成
↓ 消除目标图块 / 达成关卡目标
↓ 关卡结算（星级评定 / 材料掉落 / 光能积累）
↓ 返回輝島世界地图 → 章节解锁 / 下一关

2. 元循环（Meta Loop）

局外周期：单次修复动画 30 秒 – 2 分钟，区域完整修复跨越数天游玩。

关卡 → 材料 → 修复 → 叙事 → 下一章节
通关关卡
↓ 获得材料掉落（浮木/绳索/花种/陶片/温泉石等，对应章节主题）+ 光能积累
↓ 材料自动轻合成（无复杂配方，系统自动聚合同类材料）
↓ 区域修复进度条推进
↓ 达到阶段阈值 → 触发渐进式修复动画（4阶段之一）
↓ Glow图块光能积累满阈值 → 触发区域"点亮"视觉特效
↓ 章节完整修复 → 解锁 NPC 标志性时刻动画 + 丸山渉记忆日志碎片 + 下一章节入口
↓ 解锁新关卡包 → 新图块种类 / 新棋盘主题 / 新障碍类型

3. 会话设计原则（Session Design）

步数制（Currency: Steps） 无倒计时（No Timer） 无体力系统（No Energy） 失败零惩罚（No Resource Loss） 禅境模式（Zen Mode）
  • 步数制，非时间制：每关有步数上限，压力来自策略而非焦虑——就像律在岛上慢慢生活，没有截止日期
  • 连线自由度：路径可弯折绕行（最多2次转弯），非直线限制，增加策略空间
  • 即时反馈：每次连接触发专属音效和粒子效果，形成上瘾性微奖励循环
  • 失败宽容：步数耗尽仅回到关卡起点，不扣材料、不消耗体力——岛不会因为你慢一点就放弃你
  • 禅境出口：每章2–3个禅境关，关闭步数上限和失败判定，提供零压力消除体验
  • 无体力门控：随时进出，无需等待体力恢复，真正自由游玩节奏
维度设定设计意图
局内货币步数（Steps）策略压力而非时间焦虑
最短会话30 秒碎片时间友好
典型会话90 秒单关完成感
最长会话3 分钟挑战关上限
计时器无消除时间焦虑
体力系统无兑现治愈承诺
失败惩罚无资源扣除岛不放弃玩家

4. 连击系统（Combo System）

参考手感：天天爱消除——每次成功匹配有清晰爽快的视觉+音效反馈，连击链产生渐进增强的爆炸/粒子特效。

连续匹配之间间隔 < 2000ms 视为同一连击链，超时或无可用匹配对时重置。

5. 四种特殊图块（Special Blocks）

图块 ID 触发方式 效果 视觉表现 动画时长
炸弹（Bomb） bomb 3连击自动生成 / 关卡预置 以炸弹块为中心的 3×3 范围内所有图块消除（含障碍物及覆盖层） 橙色闪光球体，爆炸粒子+冲击波环扩散，3×3范围逐格飞散 0.5 s
风车（Windmill） windmill 4连击自动生成 / 关卡预置 清除风车块所在整行+整列所有图块（十字消除，同时触发） 旋转风车纹理，横竖双光柱同时扫过，图块向两侧飞散 0.4 s
灯光（Light / Lantern） lantern 仅关卡预置（不可连击生成） 自动寻路连接距离最近的同类型图块，无需玩家操作，连接成功后双方消除 橙色光晕持续脉冲，发光连线高速飞向目标，双方光点爆发消除 0.3 s
海浪（Wave） wave 仅关卡预置（不可连击生成） 重新随机排列棋盘上所有图块位置（保持各类型图块数量不变）；重排后无可用匹配则再次重排，最多3次 蓝色水波纹从中心扩散，全棋盘漩涡式上扬后重新落下，配合哗海浪音效 0.8 s

所有特殊块激活动画不可跳过（0.3–0.8s），保持节奏感。障碍物被清除时有专属碎裂/断链粒子效果。

7. 玩家动机循环（Player Motivation Loop）

基于 Bartle 玩家分类与自我决定理论（Deci & Ryan），Glow Island 针对核心受众设计了双轴动机引擎。

Bartle Type — Explorer

探索者：治愈体验驱动

岛屿修复的渐进视觉变化、丸山渉记忆日志的碎片解锁、每章色调与叙事的情绪递进——让探索型玩家持续有"下一个发现"的期待感。

设计响应：每4–8关触发一次可见视觉变化；修复动画总有一个温柔小细节（一只猫、晾衣绳上多了一件衣服）。

Bartle Type — Achiever

成就者：岛屿修复完成感

星级评定、剩余步数奖励、连击乘数系统——让成就型玩家在每局都有具体可量化的目标，并通过章节进度条感受宏观成长。

设计响应：3星通关 = 剩余步数≥40%上限；连击×3.0触发全屏爽快感；每章完整修复解锁标志性NPC时刻动画。

SDT — Autonomy（自主）

自主感：无门控自由游玩

无体力系统、无倒计时、随时进出、禅境模式自由选择——玩家的节奏由玩家掌控，治愈承诺从设计层面兑现。

核心功能：F-004 无体力自由游玩；禅境关每章2–3个；步数制不强迫速度。

SDT — Competence（胜任）

胜任感：平滑难度曲线

前15关纯净入门，每关引入1个新概念；4种特殊块赋予策略感；失败不惩罚资源，降低挫败感，让玩家在每局都能找到"我做到了"的时刻。

失败设计：连续2次失败内不弹出购买提示；第3次失败后场景化非侵入提示助攻器。

情感循环完整路径

律抵达輝島（到来·悠闲）
↓ 每局连连看 → 连击爽快感 → 材料掉落
↓ 修复进度推进 → 岛屿渐进视觉变化（玩家感知"我让它改变了"）
↓ NPC时刻动画 + 渉的记忆日志解锁 → 情感共鸣
↓ 律与ひなた情感线一点点清晰 → 叙事驱动进度
↓ 第6章灯塔点亮 → 律与岛互相唤醒
↓ 玩家情感成就感 → 留存 + 分享 + 复玩
