v4.0 修订说明

旧系统（炸弹、行消、自动连、重排作为棋盘图块）全部移除。炸弹/行消消除单格导致奇偶失衡的问题从根源解决：新5种特殊图块由设计本身保证奇偶安全，无需引擎运行时自动修正。旧重排和自动连功能移入UI层，成为花费沙滩币的场外辅助道具。

1 核心机制概述

Glow Island 的关卡玩法是节奏驱动的连连看（Combo-driven Mahjong Connect）。核心匹配规则与传统连连看一致：选中两个相同图块，连接路径最多转弯2次即可消除。叠加Combo计时器——快速连续匹配可维持连击链，音效节奏随连击加速，产生心流式爽快消除感。

选关 → 观察棋盘 → 配对消除 → Combo累积 → 特殊块触发 → 关卡结算
进入关卡 → 棋盘由偶数个图块构成（每种类型数量均为偶数，保证全部可配对消除）
↓ 选择两个相同图块，路径≤2次转弯，成功则消除，消耗1步
↓ 消除触发即时反馈（音效 + 粒子特效）并将Combo计时器重置为2000ms
↓ 在计时器归零前继续消除 → Combo链累积 → 倍率上升 → 音效节奏加快
↓ Combo=3生成连锁 / Combo=4生成光链 / Combo=6生成光波（重磅奖励）
↓ 特殊图块激活 → 战术效果触发 → 奇偶天然安全（无需自动修正）
↓ 所有图块消除完毕 → 关卡通关 → 材料奖励 + 叙事解锁

基础配对规则

  • 同种图块才能配对，不同图块不可连接
  • 连接路径最多转弯2次（L型或Z型路径），路径不能穿越其他图块
  • 特殊障碍（冰封块、锁链等）阻断路径，需先清除才能连通
  • 棋盘生成时保证每种图块数量为偶数；特殊图块激活不破坏此保证

4 特殊图块——5种战术型棋盘图块

v4.0 奇偶安全原则（大幅简化）

新5种特殊图块由设计层面天然保证奇偶安全，无需引擎运行时自动修正：光波仅清除障碍层（不触碰匹配图块）；光链只消除完整配对（始终偶数消除）；穿透不移除任何图块；置换只交换位置不增减图块；连锁只自动匹配合法图块对（始终偶数消除）。奇偶校验仅在关卡生成时执行一次即可。

🌟
光波（Wave）
id: wave  ·  图标：涟漪光环，暖白色
Combo=6 自动生成 关卡预置
触发条件
将两个光波图块配对连线时激活。
Combo 交互
激活计入Combo链（+1次），重置计时器。
效果说明
清除以两个光波连线中点为中心的 3×3范围内所有障碍层（冰封块、锁链、藤蔓、木箱等）。匹配图块本身不被移除，只清除障碍层。同时赋予「自由路径区」5秒增益：在已清除区域内，连线可自由穿越任意图块（忽略占据限制）。
奇偶安全说明
障碍物不是配对图块——清除障碍对任何图块的数量奇偶性均无影响。匹配图块完全不被触动。天然零奇偶风险，无需任何修正逻辑。
视觉特效
暖白色涟漪光环向外扩散 → 3×3范围障碍物碎裂消散 → 清除区域持续5秒柔和光晕（提示自由路径增益期）→ 5秒后光晕渐退。音效：清澈钟鸣声 + 障碍碎裂声 + 持续低沉环境光音。
⛓
光链（Light Chain）
id: light_chain  ·  图标：链式光环，金色连环
Combo=4 自动生成 关卡预置
触发条件
将两个光链图块配对连线时激活。
Combo 交互
每次自动匹配的配对各计入Combo链（+1次）。多对同时消除时Combo可急速飙升。最后一次自动配对完成后重置计时器。
效果说明
扫描全棋盘，找出当前数量最多的图块类型，然后自动匹配该类型所有可组成的配对——连线从所有实例同时射出，飞向各自配对目标，同时消失。若该类型数量为奇数，最后一个单块不被消除（保持偶数消除原则）。
奇偶安全说明
只消除完整配对（成双消除），始终移除偶数数量图块。奇数个时留下最后1块，不破坏奇偶性。天然安全，无需修正。
视觉特效
金色光环脉冲两下 → 扫描波横扫棋盘 → 目标类型所有图块同时高亮为金色 → 连线同时从各实例射出飞向配对目标 → 链式消失配合渐进加强的连续音效。音效：金色扫描音 + 连续「叮」声（每对一声，节奏感强）。
⚡
穿透（Pierce）
id: pierce  ·  图标：光箭，蓝白渐变，穿透感
仅关卡预置（战略性谜题元素）
触发条件
将两个穿透图块配对连线时激活。
Combo 交互
激活计入Combo链（+1次），重置计时器。穿透模式期间正常配对各自计入Combo。
效果说明
赋予玩家「穿透模式」增益，持续8秒。在穿透模式下，连线路径可穿越棋盘上任意已占据的格子（包括匹配图块和障碍物）。同种图块匹配规则不变——仍需连接相同类型图块，只是路径路由限制临时解除。
奇偶安全说明
穿透模式不移除任何图块，只修改路径路由规则。奇偶完全不受影响。
视觉特效
蓝白光箭爆发 → 屏幕四角持续蓝色边缘光晕（增益存续标志）→ 增益期间的连线呈现为发光光束而非普通连线 → 8秒后边缘光晕渐退，连线恢复正常样式。音效：穿透激活的锐利音效 + 穿透连线时的特殊穿梭音。
🔀
置换（Swap）
id: swap  ·  图标：双向箭头，绿色旋转
仅关卡预置（高技巧战术元素）
触发条件
将两个置换图块配对连线时激活，进入「选择模式」。
Combo 交互
配对置换图块计入Combo链（+1次），重置计时器。置换操作本身不额外计为一次配对。
效果说明
玩家在棋盘上选择任意两个格子——它们的内容（图块/障碍/空格）互换位置。可交换：图块+图块（同种或不同种）、图块+障碍、图块+空格。完成选择后立即交换，游戏恢复正常。玩家有10秒完成选择；若超时未选择，置换图块保留在棋盘上，等待玩家再次配对触发。
奇偶安全说明
只交换位置，不增减任何图块。总计数不变，奇偶完全不受影响。
视觉特效
两个绿色旋转螺旋出现在被选中的格子上 → 两个螺旋相向加速旋转 → 图块互换位置时产生弹性「咔哒」特效 → 绿色光晕渐退。棋盘角落显示10秒选择倒计时进度条。音效：选中音 + 置换完成的满足感「咔哒」声。
🔥
连锁（Cascade）
id: cascade  ·  图标：连环波纹，橙红渐变
Combo=3 自动生成 关卡预置
触发条件
将两个连锁图块配对连线时激活。
Combo 交互
每次自动配对各计入Combo链（+1次）。完整5连锁 = 瞬间+5 Combo，极具爽快感。最后一次自动配对完成后重置计时器。
效果说明
引擎自动执行连锁配对——找到当前路径最短的合法配对，自动连线消除，再从新棋盘状态找下一个最短合法配对，持续进行。终止条件：完成5次自动配对，或棋盘已无合法配对。每次自动配对之间有0.3秒延迟，让玩家看清连锁过程。
奇偶安全说明
只执行路径可连通的合法配对（完整图块对），始终偶数消除。不产生孤立块，天然安全。
视觉特效
橙红波纹向外扩展 → 引擎锁定第一目标对（聚光灯效果高亮）→ 自动连线消除 → 0.3s后锁定下一对 → 循环往复，节奏逐渐加快，音效强度逐渐增强，形成加速连击感。音效：连锁启动低沉音 + 每次自动消除节奏递进的清脆音。

5 场外辅助——3种沙滩币消耗道具

场外辅助说明

场外辅助不占用棋盘格子，通过UI按钮触发，是玩家遇困时的主动求助手段，非自动机制。花费沙滩币（游戏内货币），不影响棋盘图块奇偶性。

🌊
重排
30 沙滩币
触发：引擎检测到无合法配对时弹出引导 / 玩家随时主动点击
打乱棋盘上所有图块的位置，保证新布局中至少存在一个合法配对。图块种类数量完全不变，仅位置改变。随机打乱后无配对则最多重排3次直到有配对。

高亮显示当前棋盘上最优的下一步连接序列（综合Combo延续和步数效率的最佳配对路径）。高亮持续3秒后自动消失。
