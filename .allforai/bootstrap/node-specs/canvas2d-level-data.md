# canvas2d-level-data

## Mission

生成全量180关关卡配置数据，按章节难度曲线设计步数和棋盘参数。

## 关卡数据结构

```js
export const LEVELS = {
  "1-1":  { steps: 80, cols: 6, rows: 8, types: 3, specials: [] },
  "1-5":  { steps: 70, cols: 6, rows: 8, types: 4, specials: [{ type: 6, count: 1 }] },
  "1-30": { steps: 55, cols: 8, rows: 8, types: 5, specials: [{ type: 6, count: 1 }, { type: 7, count: 1 }] },
  // ... 全部180关
};

export function getLevel(chapter, level) {
  return LEVELS[`${chapter}-${level}`] || null;
}
```

## 参数说明

| 字段 | 类型 | 说明 |
|------|------|------|
| steps | number | 步数上限（每消一对扣1步） |
| cols | number | 棋盘列数（6-10） |
| rows | number | 棋盘行数（8-12） |
| types | number | 使用图块类型数量（3-5） |
| specials | array | 预置特殊图块 [{type: 6-10, count: 1-3}] |

## 难度曲线设计

### 第1章：码头（入门）
- 关1-10：6×8棋盘，3种类型，步数80-70，无特殊图块
- 关11-20：6×8棋盘，4种类型，步数65-55，引入光波(6)
- 关21-30：8×8棋盘，4种类型，步数55-45，光波+光链(7)

### 第2章：陶艺（上手）
- 关1-10：8×8棋盘，4种类型，步数60-50，光波+光链
- 关11-20：8×10棋盘，5种类型，步数55-45，+穿透(8)
- 关21-30：8×10棋盘，5种类型，步数50-40，+置换(9)

### 第3章：花田（熟练）
- 全章：8×10棋盘，5种类型，步数50-38
- 引入连锁(10)，特殊图块组合出现

### 第4章：森林（挑战）
- 全章：10×10棋盘，5种类型，步数45-35
- 特殊图块2-3个同时出现

### 第5章：温泉（高难）
- 全章：10×10棋盘，5种类型，步数40-30
- 多个特殊图块，严格步数限制

### 第6章：灯塔（大师）
- 全章：10×12棋盘，5种类型，步数38-28
- 复杂特殊图块组合，最终关为Boss关（步数25）
